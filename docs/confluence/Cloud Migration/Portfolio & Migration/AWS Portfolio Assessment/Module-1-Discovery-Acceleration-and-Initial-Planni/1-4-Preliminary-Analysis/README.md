title: 1.4 Preliminary Analysis
-------------------------------

Overview
========

The objective of this group of activities is to perform a preliminary analysis of the data collected. Note that these sessions are not intended to perform a complete portfolio analysis which is rather targeted by the 

Module 2-Portfolio Analysis and Migration Planning
stage.

Pre-requisites Checklist
------------------------



Data collected and available for analysis in a suitable format
