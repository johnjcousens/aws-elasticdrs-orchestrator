8 Resources To Get Started with Foundation/CloudScape
-----------------------------------------------------

Source: [https://community.flexera.com/t5/Foundation-CloudScape-Knowledge/8-Resources-To-Get-Started-with-Foundation-CloudScape/ta-p/135011#](https://community.flexera.com/t5/Foundation-CloudScape-Knowledge/8-Resources-To-Get-Started-with-Foundation-CloudScape/ta-p/135011)

### New to Foundation/CloudScape?

We're here to help!  Below are recommended resources and a high-level overview of assessment steps.

#### Available Training:

The Flexera Learning Center includes a CloudScape learning path with a course on each assessment step listed below.  We highly recommend completing the CloudScape Learning Path before beginning your first assessment: <https://learn.flexera.com/series/cloudscape/getting-started-with-cloudscape>

#### High-level CloudScape Assessment Process and Helpful Documentation Links

1. RN150 virtual appliance deployment (and FlexDeploy appliance deployment if required)
2. [PreDeployment Checklist](https://portal.riscnetworks.com/app/documentation/?path=/getting-started/predeployment-checklist/)
3. [Quick Start Guide](https://portal.riscnetworks.com/app/documentation/?path=/getting-started/quick-start-guide/)
4. [FlexDeploy Getting Started Guide](https://portal.riscnetworks.com/app/documentation/?path=/overview/alternate-and-additional-deployment-methods/)
5. Discovery Scans for Initial Asset Inventory
6. [Discovery How To](https://portal.riscnetworks.com/app/documentation/?path=/getting-started/discovery%253A-how-to/)
7. [Discovery Troubleshooting](https://portal.riscnetworks.com/app/documentation/?path=/getting-started/discovery-troubleshooting/)
8. [Assets & Asset Errors](https://portal.riscnetworks.com/app/documentation/?path=/using-the-platform/reports%252Fpages/)
9. [Glossary- Device Types Definitions](https://portal.riscnetworks.com/app/documentation/?path=/using-the-platform/glossary/)
10. Apply Licensing for Performance Collection
11. If using Burst licenses see Community article:
12. [Best Practices for Using Burst Licenses](https://community.flexera.com/t5/Foundation-CloudScape-Knowledge/Best-Practices-for-Using-Burst-Licenses/ta-p/125973)
13. Review Performance Collection
14. 24-48 hours after applying Licenses, review potential collection errors in 
    *Collect Data > Summary of Collection* 
    page in assessment
15. Application Stack rationalization and confirmation
16. Approximately one week after licensing, run "Build App Stacks"
17. [Building Application Stacks](https://portal.riscnetworks.com/app/documentation/?path=/using-the-platform/building-application-stacks/)
18. Add relevant application stack tags
19. Begin leveraging available reports and platform pages
20. [Reports/Pages](https://portal.riscnetworks.com/app/documentation/?path=/using-the-platform/reports%252Fpages/)
