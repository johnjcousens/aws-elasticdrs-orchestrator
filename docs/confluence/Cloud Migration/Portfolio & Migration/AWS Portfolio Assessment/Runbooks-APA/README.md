There are many services and partner tools available for Portfolio Discovery. In addition, customers may have implemented tools (e.g., App performance tooling, CMDBs) that they can utilize to extract and provide data for analysis by consultants.

This delivery kit includes the following base runbooks for the most common discovery tooling and is intended as a guide to implementing these tools. The selection of a discovery tool is covered in [Module 1-Discovery Acceleration and Initial Planning](../Module-1-Discovery-Acceleration-and-Initial-Planning)
