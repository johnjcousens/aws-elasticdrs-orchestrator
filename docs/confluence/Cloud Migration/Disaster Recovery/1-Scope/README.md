**Appendix K**

**Disaster Recovery**

**Regional Disaster Recovery Assessment**

AWS will provide consulting and advisory services described below:

Assist Customer with discovering and assessing Customer’s current On-Prem or Cloud state across data storage, infrastructure, operations, network, and security capabilities for Guiding Care application Assist Customer with compiling and confirming a list of per-requisites required for Disaster Recover("DR") implementation.

Perform gap assessment of Customer’s current infrastructure and application capabilities against high availability business requirements and AWS disaster recovery best practices.

Assist Customer with building a DR plan including database, application, infrastructure, network and operations

Deliverables:

Document assessment report of existing infrastructure and application

Document AWS Disaster Recovery options and strategy recommendations

Document high level Disaster Recovery roadmap on AWS
