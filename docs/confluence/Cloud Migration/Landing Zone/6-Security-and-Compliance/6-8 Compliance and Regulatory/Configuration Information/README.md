**Purpose**  
This page serves as the centralized summary for all configuration information related to the organization’s HITRUST CSF domains. It consolidates references to the detailed configuration documentation contained within the child pages for each domain family, ensuring complete visibility, traceability, and stakeholder approval.

**Scope**  
This summary covers all HITRUST CSF domains, with each domain’s configuration information maintained in its respective child page. The information contained herein and in the linked child pages represents the finalized, approved configurations in place for systems handling regulated or sensitive information, including PHI, PII, and other classified data types.

**Objective**  
To ensure that:

* All configuration settings for each HITRUST CSF domain are documented, reviewed, and approved.
* Configurations are aligned with policy, compliance, and regulatory requirements.
* A single authoritative record is maintained for audit readiness and operational reference.
