**Related AWS documentation**

* Landing Zone Accelerator on AWS is a fully-automated implementation of the architectural guidelines documented in the [AWS Security Reference Architecture (SRA)](https://docs.aws.amazon.com/prescriptive-guidance/latest/security-reference-architecture/welcome.html).
* Landing Zone Accelerator on AWS incorporates features and lessons learned from previous accelerator solutions such as the [Compliant Framework for Federal and DoD Workloads in GovCloud (US)](https://aws.amazon.com/solutions/implementations/compliant-framework-for-federal-and-dod-workloads-in-aws-govcloud-us/) and the [AWS Secure Environment Accelerator](https://aws-samples.github.io/aws-secure-environment-accelerator/)
