**Purpose**
-----------

Document the decision for the landing zone SSO federation.

**Decision**
------------

HealthEdge has decided to configure IAM Identity Center to utilize their Okta Identity Provider (IdP) for authenticating user SSO access to AWS accounts. The details of the SSO federation design are documented in AWS-Federation-for-landing-zone .
